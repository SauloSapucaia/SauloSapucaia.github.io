Dashboard Operacional

Análise de produtividade, prazos e gargalos para otimizar a eficiência de processos operacionais.

Power BI DAX Power Query SQL

Contexto e Problema

A organização possuía um grande volume de processos operacionais, mas a gestão era baseada em relatórios manuais e planilhas fragmentadas. Isso dificultava a análise de produtividade, o acompanhamento de prazos e a identificação de gargalos.

Dashboard interativo em Power BI. Entre os benefícios:

Pipeline End-to-End: APIs, Azure Fabric e Monitoramento

Desenvolvimento de pipeline completo com Python, bancos de dados no Azure (Fabric) e Power Automate para processamento de JSON e controle via Power BI.

Python Azure Fabric Power Automate Power BI

Contexto e Problema

O fluxo dependia de consultas manuais em sistemas externos para atualizar documentações operacionais. Era necessário processar dados de formulários para identificar com precisão a autoria dos processos, distinguindo automaticamente qual usuário foi o responsável pela tratativa, e garantir a notificação imediata das partes envolvidas, tudo de forma escalável, segura e rastreável.

Processo e Solução

  • Integração e Pipeline: Criação de pipeline em Python para consumo de APIs REST de terceiros, realizando a extração e o processamento complexo de payloads no formato JSON.
  • Cloud & Engenharia de Dados: Estruturação e gravação das informações processadas em um banco de dados hospedado no Azure, orquestrado através do Microsoft Fabric.
  • Automação de Comunicação: Integração direta com o Power Automate para o roteamento inteligente e disparo automatizado de e-mails customizados baseados nas regras de negócio extraídas da API.

Entregáveis e Resultados

Um ecossistema de dados automatizado e robusto que entregou:

  • Eliminação total de consultas sistêmicas repetitivas e clareza na rastreabilidade de responsabilidades da equipe.
  • Dashboard de Governança: Criação de um painel interativo no Power BI, consumindo os dados diretamente do banco no Azure, permitindo monitorar o envio dos processos de acionamento e acompanhar o controle de execução e saúde do programa em tempo real.
