## 🚀 Projetos em Destaque

Neste portfólio, detalho 5 estudos de caso reais aplicados ao negócio, divididos em duas frentes:

### 📈 Business Intelligence
1.  **Dashboard Operacional:** Análise de produtividade, acompanhamento de SLAs e identificação de gargalos logísticos/operacionais com modelagem Star Schema e ETL via Power Query.
2.  **Dashboard de Atendimento:** Monitoramento de tickets de suporte, reduzindo o acúmulo de chamados atrasados com consolidação de dados via Python e Power BI.
3.  **Dashboard Financeiro:** Visão consolidada de custos operacionais e margens, permitindo a comparação rápida de eficiência entre diferentes unidades de negócio.

### 🐍 Python & Automação
4.  **Ajuste de Métrica de Performance:** Refino de regras de negócio complexas via Python para refletir a realidade operacional, garantindo validação analítica confiável.
5.  **Automação de Input de Relatórios (RPA):** Script de Web Automation desenvolvido para realizar upload de documentos e inputs em sistemas terceiros, eliminando falhas manuais e ganhando produtividade.
